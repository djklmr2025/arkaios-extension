🚀 ARKAIOS Lab – Chat Module

Extensión para Chrome / Edge que integra ARKAIOS Lab directamente con un chat lateral (overlay).

📦 Descargar e instalar (.crx) 📥 Descargar ZIP (Stable) 🔧 Descargar ZIP (Debug)

Cómo instalar el .crx:
1. Ve a chrome://extensions o edge://extensions
2. Activa “Modo desarrollador”
3. Arrastra y suelta el archivo `.crx` dentro de la página
4. Confirma la instalación

O bien, usa el archivo ZIP: descomprímelo y selecciona la carpeta desde “Cargar extensión sin empaquetar”.
